# 📜 Privacy

**Privacy** is a Dalamud plugin for Final Fantasy XIV that gives players a cleaner and more personal way to manage contacts, private notes, favorite venues, online profiles, status information and quick social actions.

## ✨ Features

### 🔐 Private Contact List

Create your own private list of players with useful information attached to each entry.

- Add players to your private list
- Search contacts quickly
- Show or hide offline contacts
- See contact locations when available, including housing information
- Keep useful contact details such as nicknames, tags, colors, notes and images
- Open contact profiles with extra personal details
- Pin important contacts so they stay easier to find
- See visible indicators for pinned contacts
- Contact action buttons remain available even when a contact is offline

### 🖱️ Right-Click Privacy Actions

Privacy adds helpful options to the native in-game player context menu. Depending on the target, you can:

- Add a player to your private list
- Remove a player from your private list
- Open a Privacy Online Profile when the target is a known Privacy user

### 📌 Favorites and Pinned Contacts

Favorites and pins help you keep important people easy to access

- Mark contacts as favorites
- Pin contacts directly from the contact row
- View important contacts more easily
- Receive notifications for favorite contacts
- Optionally limit notifications to favorites only
- Use favorites or pins for close friends, party members, roleplay contacts, venue staff or regular visitors

### 🔧 Quick Contact Actions

Saved contacts can include quick action buttons for common interactions such as:

- Send a tell
- Invite to party
- Travel or teleport when supported
- Open estate teleportation options
- Set custom tags
- Open or edit private notes

### 📝 Notes

Privacy includes a simple notes system for anything, examples:

- How you met someone
- Roleplay details
- Preferred content
- Personal reminders
- Venue details
- Labels or context you want to remember later

Notes can also appear as helpful tooltips in the main list.

### 👥 Contact Profiles

Each saved contact can have extra profile details saved locally in the plugin.
Supported details include:

- Nickname
- Custom symbol
- Main job
- Role or relationship note
- Nameday or birthday
- Preferred content
- Bio
- Custom color
- Status history

These fields are for your own organization and do not change the player's actual in-game profile.

### 📷 My Profile

Privacy includes a personal profile window where you can customize how your own Privacy profile appears.
You can set:

- Display name
- Display name effects
- Status message
- Bio
- Profile image
- Banner image URL
- Avatar border
- Status color
- Favorite venues
- Profile theme
- Theme color

The Profile Preview updates live so you can see how your Online Profile will look before sharing it.

### 📲 Online Profiles

When available, Privacy can show profile information shared through the plugin's online profile system.
Online Profiles can include:

- Display name
- Animated display name effects
- Custom status
- Bio
- Profile image
- Banner image
- Favorite venues
- Selected profile theme
- Theme color
- Avatar border
- Public location preview when allowed

Profile themes and visual effects are visible to other users through Online Profiles.

### 🎨 Profile Themes

Privacy includes profile themes for Online Profiles and Profile Preview.
Available profile themes include:

- Casino
- Frostbite
- Neon Grid
- Pastel Bloom
- Glass Rain
- Minimal Ink
- Royal Sigil
- Ember Forge
- Ocean Tide
- Void Stars
- Gold Saucer

Profile themes can include custom colors, decorative overlays, visual effects and unique section icons.

### 📑 Display Name Effects

Your display name can use visual effects in My Profile and Online Profiles.
Effect categories include:

- Wave
- Pulse
- Static

Display name effects update live in My Profile and Profile Preview and can appear publicly in your Online Profile.

### 🔑 Profile Privacy Controls

Privacy lets you choose what other users are allowed to see from your Online Profile.
You can control visibility for:

- Status
- Venues
- Location
- Bio

Each visibility option can be set to:

- All
- Friends
- Nobody

This lets you share only the information you are comfortable showing.

### 💡 Status and Presence

Privacy can show useful online, offline and activity information.
Supported status features include:

- Custom plugin status
- Role-Playing status
- Auto Status by Activity
- Native in-game status icons
- Online and offline detection for contacts using Privacy accounts
- Notifications when contacts come online or go offline

Auto Status by Activity can update your plugin status based on your in-game state such as Busy, AFK, Content or Role-Playing.

### ⛪ Venues

Privacy includes a dedicated venue list for saving and organizing places where you can:

- Save venues
- Search venues
- Mark venues as favorites
- Add venue notes and tags
- Add profile images for venues
- Save venue colors
- Add role information if you work at a venue
- Open venue Discord links when available
- Check opening or closing information when available
- Use teleport commands for saved venues

### 🔍 Discover

The Discover view helps you import or manage players from your in-game friend list where you can:

- View native friends loaded from the game
- Filter by online, offline or not added yet
- Add online friends
- Add only friends not already listed
- Add all currently shown friends

If no friends appear, open the game's Friend List once and then return to Privacy.

### 🗂️ Groups

Groups help organize contacts into custom sections.
You can use groups for anything, for example:

- Static parties
- Roleplay circles
- Venue staff
- Friends by Data Center
- Content groups
- Personal categories

## 🔔 Notifications

Privacy can notify you about useful contact activitys including:

- Show online count on login
- Notify when favorite contacts are online
- Limit notifications to favorites
- Track contact status history
- Keep last known locations for offline contacts
- Avoid false duplicate online/offline notifications when possible

### 🗝️ Keybinds

Privacy supports configurable keybinds for common actions.
You can set keybinds for:

- Opening the main Privacy window
- Opening the target's Online Profile
- Opening target notes

Pressing the same keybind again closes the related window if it is already open. Pressing **ESC** while recording a keybind cancels the recording.

### 🌈 Customization

Privacy includes several visual customization options such as:

- Accent color
- Window background color
- Top and bottom bar colors
- Contact row background
- Venue row background
- Dividers
- Pre-made theme presets
- Main window background image
- Background image effects
- Built-in image suggestions

Available UI presets include styles such as red, gold, cyberpunk, water, fire, sakura, frost, autumn and more.

## 🛠️ Commands

| Command | Description |
| --- | --- |
| `/privacy` | Opens or closes the main Privacy window. |
| `/plist` | Opens or closes the main Privacy window. |
| `/list` | Opens or closes the main Privacy window. |
| `/plistaccount` | Opens the account/login window. |

## 🔓 Basic Usage

1. Open Privacy with `/privacy`, `/plist`, or `/list`
2. Add players through the right-click menu or from the Discover view
3. Use Favorites, Pins, Groups, Notes and Tags to organize your list
4. Save venues you visit often in the Venues section
5. Customize your own profile in My Profile
6. Open Privacy Settings to adjust visuals, notifications, keybinds and privacy controls

## 📥 Installation

Privacy is distributed as a custom Dalamud plugin.
Add the plugin repository to Dalamud, then install **Privacy** from the plugin installer.

Repository URL:

```txt
https://raw.githubusercontent.com/seventity7/Privacy/main/repo.json
```

## 📄 Notes

- Privacy is an unofficial plugin and is not affiliated with Square Enix
- Some actions depend on the game UI and may require the related game window or system to be available
- Venue information depends on available data and may not always be complete
- Private notes, nicknames, colors and local organization fields are meant for your own plugin display
- You do not need to create an account to use the local Privacy features
- Online Profile features require a Privacy's account
