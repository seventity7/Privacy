# Privacy

**Privacy** is a Dalamud plugin for Final Fantasy XIV that adds a clean private list for players, contacts and venues.

It is made for players who want a more personal way to keep track of friends, favorite places, notes, status, profiles and quick social actions without relying only on the default game UI.

---

## Features

### Private Contact List

Keep your own private list of players with useful details attached to each entry.

- Add players to your private list
- Mark contacts as favorites
- Search contacts quickly
- Show or hide offline contacts
- See in real time where your contacts are _(including housing ward/plots etc)_
- Keep the last known location while a contact is offline
- Add nicknames, symbols, tags, colors, notes and profile images
- Open a contact profile window with extra personal details

---

### Right-Click Add to Privacy

Privacy adds a custom option to the game's player right-click menu.
When enabled, you can right-click a visible player and choose:

> **Add to privacy**

This makes adding players faster and avoids typing their information manually.

---

### Favorites

Favorites make important contacts easier to find.

- View favorite contacts in a dedicated section
- Receive automatic notifications for favorite contacts
- Optionally limit status notifications to favorites only
- Use favorites to keep close friends, party members, roleplay contacts or regular visitors easy to access

---

### Quick Contact Actions

Each saved contact can show quick action buttons for common tasks.

- Send a tell
- Invite to party
- Travel or teleport toward the contact location when supported
- Open estate teleportation options
- Set custom tags
- Set private notes

---

### Notes

Privacy includes a simple notes system for contacts and venues.
Use notes for things like:

- How you met someone
- Roleplay details
- Preferred content
- Reminders
- Personal labels or context
_& etc.._

Notes are also used as helpful tooltips in the main list

---

### Contact Profiles

Each contact can have extra profile details saved locally in the plugin.
Supported fields include:

- Nickname
- Custom symbol
- Main job
- Role or relationship note
- Nameday or birthday
- Preferred content
- Bio
- Custom color
- Status history

These fields are for your own organization and do not change the player's actual game profile.

---

### My Profile

Privacy includes a personal profile window for your own character where you can set:

- Display name
- Status message
- Bio
- Profile image
- Status color
- Favorite venues _(Also show they address and info)_

---

### Online Profiles

When available, Privacy can show profile information shared through the plugin's online profile system. _(You need a account)_
This can include:

- Display name
- Custom status
- Bio
- Precise location preview
- Favorite venues
- Own profile image

---

### Venues

Privacy includes a dedicated venue list for saving and organizing places where you can:

- Save venues _(You can also show your Role if you work there)_
- Search venues _(you can choose a venue from a list that contains all NA venues)_
- Add profile images for venues
- Mark venues as favorites
- Add notes and tags
- Save venue colors
- Open venue Discord links when available
- Check opening or closing information when available
- Use teleport commands for saved venues

---

### Discover

The Discover view helps you import or manage players from your game friend list.
You can:

- View native friends loaded from the game
- Filter by online, offline or not added yet
- Add online friends
- Add only friends not already listed
- Add all currently shown friends

If no friends appear, open the game's Friend List once and then return to Privacy.

---

### Groups

Groups help organize contacts into custom sections.
You can use groups for thing like:

- Static parties
- Roleplay circles
- Venue staff
- Friends by data center
- Content groups
- Personal categories

---

### Status and Notifications

Privacy show useful login and status information.
Options include:

- Show online count on login
- Notify when favorite contacts are online
- Limit notifications to favorites
- Track contact status history
- Keep last known locations for offline contacts

---

### Customization

Privacy includes several visual customization options.
You can customize things like:

- Accent color of UI
- Windows background color
- Top and bottom bar colors
- Contact row background
- Venue row background
- Dividers
- Use pré-made theme presets
- Custom image for the main window background
- Background image effects
- Choose custom images from built-in image suggestions

Available theme presets include styles such as red, gold, cyberpunk, water, fire, sakura, frost, autumn and more.

## Commands

| Command | Description |
| --- | --- |
| `/privacy` | Opens or closes the main Privacy window. |
| `/plist` | Opens or closes the main Privacy window. |
| `/list` | Opens or closes the main Privacy window. |
| `/plistaccount` | Opens the account/login window. |

---

## Basic Usage

1. Open Privacy with `/privacy`, `/plist`, or `/list`
2. Add players through the right-click menu or from the Discover view
3. Use Favorites, Groups, Notes and Tags to organize your list
4. Save venues you visit often in the Venues section
5. Open Privacy Settings to customize the look and behavior of the plugin

---

## Installation

Privacy is distributed as a custom Dalamud plugin.

Add the plugin repository to Dalamud, then install **Privacy** from the plugin installer.

Repository URL:

```txt
https://raw.githubusercontent.com/seventity7/Privacy/main/repo.json
```
